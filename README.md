# FunnelLab  
### Powered by GMHCO LTD

FunnelLab is a performance-focused digital system built to help businesses improve their advertising results through structured Google Ads setup, conversion tracking, and data-driven funnel optimisation.

This project is designed as a lightweight, deployable web presence with integrated lead capture and backend automation support.

## 📊 Features

### Frontend
- Modern neon-flow UI design  
- Responsive layout (mobile + desktop)

### Backend (optional setup)
- Secure form submission API  
- Email automation system  
- Lead routing to inbox  
- Scalable serverless architecture  

---

## 📩 Lead System Workflow

1. User submits enquiry form  
2. Data is sent to backend API  
3. Backend processes lead securely  
4. Email notification is sent to owner  
5. Lead is stored for follow-up (optional CRM integration)
